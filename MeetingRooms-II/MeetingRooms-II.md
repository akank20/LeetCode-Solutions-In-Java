Given an array of meeting time intervals ```intervals``` where ```intervals[i] = [starti, endi]```.
Return the minimum number of meeting rooms required.


**Sample Input 1:**
```
Input: 
intervals = [[0,30],[5,10],[15,20]]
Output: 2
```

**Sample Input 2:**
```
Input: 
intervals = [[7,10],[2,4]]
Output: 1
```
